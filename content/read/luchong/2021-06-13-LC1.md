# 1.背景
一种语言的生命力受一系列语言意识形态的影响，为语言的某些方面指定数字分数以确定其生命力的等级。语言意识形态是支撑人们如何进行互动、交流和评估语言活动的信念，包括关于一个语言的优势和劣势的信念，以及关于手语和手势的语言地位的信念。语言意识形态被定义为关于语言的“潜在的、未表达的、潜意识的想法”，包括语言的用法、价值、起源、合理化、理由和未来轨迹。21世纪的意识形态更侧重于功能、意义和语用，以及挑战语言假定boundness（有界性/局限性）的概念，如多模态和跨语言。但围绕语言生命力的意识形态研究文献主要集中在口语方面，对于手语的研究几乎是空白。本文对支持联合国教科文组织相关话语的假设进行了批判性的审视，并对手语生命力评分所涉及的意识形态进行了反思。
# 2. 支撑语言活力衡量的两种意识形态
到目前为止，关于语言濒危意识形态的文献，几乎都集中在口语上。生物多样性在很大程度上已被确立为讨论濒危语言的“概念框架”。这一框架根深蒂固，语言活力如此之强，很难将两者分开讨论，因此“生物多样性”的修辞不可避免地出现在本文中。将语言多样性定义为生物多样性的另一个原因是，世界上大多数地区的丰富的生物多样性都在土著人民的管理或保护下，他们的区域往往是使用小规模濒危语言的社区。
环保主义和语言学中的“多样性”论调为“代表明显全球化的共同利益”提供了基础。在这种修辞中，对语言的“入侵”被描绘成“像病毒一样的攻击”，“破坏它们的健康”。生物多样性框架缺乏对政治权力结构和少数群体语言权利的充分考虑，“实际上强化了它所抗议的进化变化的必然性。
“科学”方法对待濒危语言有时会促进生物多样性的概念框架，部分原因是，如果语言被视为多样性且平等的物种，我们可以对其应用相同的因素，那么将所有语言与一个尺度进行比较要容易得多。人们认为所有语言都构成了人类的全部多样性，这种试图“包容”所有语言的想法，即教育中的“包容”和“平等”意味着一视同仁。为了对濒危语言的生命力得出准确的结论，须对濒危语言进行比目前更为敏感的个体语言审查。每种语言都存在于不同的民族国家，有很大的差异性。手语具有一种特殊的韧性，是口语所没有的，部分原因是手语者对保持其语言的重要性的集体解释，但鉴于失聪儿童很少有机会从流利的成人手语手中学习手语，这种说法也可能受到质疑。
语言是一种文化的大厦这一观点可以被视为一种“保护主义论据”，它描绘了一个区域和一种传达其文化信仰的语言之间的天然联系。这常常与一种所有权思想联系在一起，即某一特定群体拥有这种语言，并保留对其进行定性判断的权利。聋人在道义上有责任提升手语的地位，少数语言群体通常认为他们的语言是文化知识的宝库，对于聋人社区来说尤其如此，因为他们的视觉文化是以手语为基础的，一种语言的消亡被认为是“对世界知识的集体丧失”，因为这种语言的结构编码了它的文化智慧。多样性和生态化的话语可能是全球主义意识形态的例证。重新断言语言作为一种特定文化的大厦的价值可能被视为对全球化的“现代主义反应”。

# 4.个案研究：尤卡特克玛雅手语活力评分背后的意识形态
以尤卡特克玛雅手语(YMSL)为例来说明问卷中的意识形态问题。YMSL是出现在尤卡坦玛雅人社区的本土手语，在墨西哥尤卡坦农村地区耳聋发病率很高。由于这些社区的聋人成员从未接触过墨西哥手语，或任何其他既定的手语，他们发展了自己的当地手语。聋人和听力正常的人都会使用。在联合国教科文组织的调查中，我们包括了Chicán村和Nohkop村的数据，Chicán村有来自不同家庭的16名聋人，Nohkop村有4名聋人兄弟姐妹。Chcan和Nohkop的YMSL都被评为“严重濒危”(2级)，因为它们的用户数量很少，手语社区持续分散，而且最近没有聋儿出生。
4.1.语言的名称 
第一场意识形态辩论涉及到YMSL是否应该被认为是一种具有统一手语不同地区变体，还是应该被认为是不同村庄的几种不同的手语。尽管来自不同区域的手语者过去没有接触过，其手语是独立出现的，但他们在不同的语言层面上表现出相似之处。相似之处可归因于这样一个事实，即手语者来自说尤卡特克语的玛雅人使用的相同传统手势，拥有相同的文化背景。
给一种语言贴标签本身就是一个意识形态过程：它界定了一种语言的边界，并将其与其他语言或变体区分开来。无论我们看的是在更大的地理区域使用的手语变体，还是专注于一种特定的语言，都决定了参考社区的大小，因此对分配给这种语言的活力得分有直接的影响。
4.2.YMSLS问卷填写中的难点问题
除了语言命名问题外，调查问卷的其他部分都很难回答YMSL的问题。一个是要求可靠地估计YMSL社区中的手语者数量。调查问卷建立在对“谁是流利/称职的手语使用者？”的直观理解基础上。这就提出受访者是否应该是评估语言使用者手语熟练度评定者，以及根据什么理由的问题。在这两种情况下，都不清楚受访者是如何确定谁是“流利的手语者”，以及他们采用了哪些标准来评估人们的手语技能。(手语)“语言能力”不是一个绝对值，而是取决于特定的交际群组和各种环境因素，在尤卡特克玛雅人社区，一些听障手语者表示，他们可以很容易地与聋人亲属沟通，但在理解其他家庭手语时却有困难。
另一个困难是，要确定一个村庄手语者社区的一代手语人是由什么构成的。在共享/村庄手语的情况下，不能只根据年龄组来划分代际，但也有必要考虑其他因素，如互动网络、教育经历和接触其他语言。对于YMSL，有人建议，在中国，区分不同的“互动群体”比区分几代手语者更有意义。因此，回答诸如“所有世代/年龄段的人，包括大多数儿童，都能熟练地使用这种语言”(问题4a；见下文图3)这类语言的问卷部分就变得有问题了。

4.3.来自语言界和政府的语言态度
在关于YMSL的公开讨论中，我们遇到了各种争议和“意识形态冲突”。从来自城市社区(例如欧洲或美国)的聋人的角度来看，chican村经常被描绘成一个“聋人乌托邦”，在那里每个人都知道手语，聋人不受歧视-但现实比这复杂得多。在不同的语境中，这种语言被描述为“一种丰富的手语”，“可能是世界上最古老的手语”，“一种仅基于常识的简单交流系统”，或者“国家文化遗产的一部分”，这取决于说话的人和背后的意识形态。
在像墨西哥这样拥有364种土著语言变体的多语种国家，语言和文化多样性被定义为一种需要保护的丰富性，它构成了一个民族遗产的一部分。墨西哥有法律保护土著少数民族语言使用者以及墨西哥国家手语使用者的语言权利。但到目前为止，YMSL或墨西哥其他地区使用的任何其他土著手语(如在瓦哈卡州或恰帕斯州，2013年)都没有得到法律承认。教科文组织的所有调查似乎都不能全面评估各国政府和语言使用者的复杂且往往相互矛盾的态度。

# 5.改编后的调查中一些因素背后的意识形态观点
5.1.代际或年龄组语言使用
在调整后的调查中，按世代或年龄组使用手语是项目4(见图3)。有人认为代际传递是防止语言衰退的最关键因素。对于手语，儿童的习得被认为是决定活力水平的重要因素。如果要在评分过程中区分因素权重，那么这一个因素的权重可能应该最大。
聋哑儿童接触成人手语的机会非常有限，这使得他们很难获得手语。这往往导致语言剥夺，这不仅威胁到语言多样性和聋人文化，也威胁到聋人儿童和成年人的认知和情感健康。识字和正规教育在农村手语社区变得越来越重要。这导致聋人和有听力的人在教育和职业方面的机会不平衡，并造成更多聋人被剥夺语言的情况，因为孩子们上的学校不提供手语。目前还不清楚这种语言剥夺的风险对手语活力有什么影响。
代际使用和传播的因素包含一种意识形态假设，即手语是有限的实体，可以代代相传。
调查中有一种假设，即语言转移是生命力的主要威胁。(问题4b的一个选项，如下面的图3所示，说明“只有少数人仍在使用手语，其他人都已转向其他语言”。)。事实上，许多聋哑儿童可能没有转向任何其他语言，但可能完全被阻止手语，例如因为他们父母的选择，医疗或教育专业人员的干预，缺乏与其他手语者的联系，和/或公开的机构政策。
5.2.政府和机构的语言态度和政策
这一因素在改编后的调查中的问题8中涵盖。当一种国家语言在法律上得到承认和制度化时，人们开始认为这种语言与进步有关，而其他语言则会过时或不需要。大多数手语都处于危险之中，因为政府的语言政策没有将它们制度化，也没有解决手语者越来越少的代际传播机会。各国政府往往通过立法，仅承认名义上的语言，或限制口译和笔译的资金，而没有采取任何其他实际措施来提供(例如，为语言课程提供资金、为聋人提供手语教学、为聋人的父母提供专门的手语教学，或减少师范教育和聋人执照的障碍)，但没有提供任何其他实际措施(例如，为聋人提供语言课程资金、手语媒介教育、为聋人父母提供专门的手语教学，或减少师范教育和聋人执照的门槛)。这剥夺了实际使用该语言的人的权利。换句话说，政府对濒危语言的态度似乎与濒危话语本身的一个潜在弱点有关：它关注的是受到威胁的语言，而不是受到威胁的社区。
5.3.社区成员对自己手语的态度
这个因素，也就是调整后的调查中的第10个问题，是受访者面临的最多方面和最具挑战性的因素。当非聋人学者代表聋人关于手语的意识形态时，这是有问题的。许多受访者和评分委员会成员，包括本文的两位作者，都在听取学者的意见，教科文组织的大多数顾问和濒危语言文学作品的作者也是如此。如果研究人员的第一语言是主导语言，他们往往对自己的特权地位和作为多数语言使用者所享有的“市场利益”知之甚少。学者们有时低估了社区成员自己对他们语言的态度的重要性，甚至可能拒绝他们记录或振兴他们语言的努力。这可能会在濒危语言使用者中助长一种自卑的思想，这种思想可能会因政治和经济环境而恶化。由于社区成员的态度过于简单化，问卷评分过程本身的一些方面也可能助长这种自卑的意识形态，即使作为评分委员会的成员，我们的意识形态目标是提请人们注意振兴或保护的必要性。
总而言之，根据我们对调整后的调查的思考，受访者认为语言的活力水平总体上略有上升的趋势，而评分委员会认为语言的活力水平较低。但是，一种语言的生命力是一种更复杂的东西，不能用数字分数来表达；一种语言可能在一个层面上濒临灭绝，但在另一个层面上却是健壮的。例如，一种手语可能在其用户社区中蓬勃发展，尽管从更大的、长期的社会政治角度来看，它是濒危的。
# 6联合国教科文组织2018年手语调查中的意识形态转变
6.1.手语社区获取数据收集和评分过程的机会
在修改后的调查工作中，一个令人担忧的问题是，数据收集和评分的方法应该最大限度地增加手语社区成员作为专家、倡导者和分析师的参与。此外，问卷答复者和评分委员会10都包括手语社区的几名聋人成员，其中一些人本身就是被认为是濒危手语的使用者。联合国教科文组织关于手语的新调查没有任何手语版本，也没有任何关于调查或调查目的的手语说明。人们越来越意识到，手语研究应该以尽可能强大的方式让聋人手语者参与，最好是由他们领导(手语语言学学会，2016年)，而且应尽可能提供研究目的、方法和成果的手语版本，以便手语者更容易获得。11首次同时发布两份调查问卷，是手语社区将其语言与口语并驾齐驱的典范。但如果口语问卷有四种口语(英语、法语、西班牙语和俄语)，那么手语调查至少提供一种手语将是有益的。这将提高手语的知名度。相反，手语调查只提供口语，所以仍然存在不平衡的情况，这对手语使用者来说是不利的。虽然我们认为，将调查问卷翻译成这四种主要语言以外的其他语言将是有利的，以便非主要语言(例如土著语言)的使用者能够进入调查，但现状提出了一个关于统一关于口语和手语活力的研究的合理性的问题。目前还不清楚研究人员和机构愿意和能够在多大程度上让说话者和签名者平等地获得评估濒危语言的过程。这反映了一种相当霸道的意识形态。改变这种意识形态可能需要很长时间，尽管我们看到了朝着这个方向迈出的步伐。
6.2.作为离散的静态实体的二进制分类的手语
由于教科文组织希望以相对标准化、易于处理的方式收集数据，因此新的调查假设手语是离散的有界实体，包含许多简单二元选择的实例，并捕获静态而不是纵向的数据，这或许是可以理解的。在调整后的调查中看到的分级和细微差别的余地，例如受访者可以为他们的每一个答案指出现有证据的强度的“可靠性指数”，在新的调查中并不存在(见图3)。在新的调查中，没有考虑到分级和细微差别，例如，受访者可以指出他们每个答案的现有证据的强度(见图3)。以自动化的方式收集数据，并确保数据易于处理是最重要的问题，它必然优先于定量数据，而不是定性数据，反映了“生物物种”的思想，即语言是可以用经验方法进行研究的实体，而且直到最近十年，我们才有任何关于手语濒危的数据。标准手语变体在双语教育项目中非常有价值，但考虑多模态对生命力的影响可能是有用的。目前还不清楚多模态对手语的活力有多大程度的保护或有害影响。
6.3.等级主义和全球主义观点
强调等级制度和政治经济考量。这再次反映了教科文组织作为一个联合国组织的性质，这是可以理解的，但有几个项目的结构背后可能存在的意识形态假设值得质疑。答案选项中意识形态等级的例子包括：将“国际”放在顶部，将“本地”放在底部(问题16)；将“第五部门--最高决策层”放在顶部，将“原材料和基本食品的生产”放在底部(问题17)；将“管理人员和专业人员”放在最高位置，将手工艺和农业工人放在较低位置(问题13)；将“公共/机构领域”放在顶部，将“私人领域”放在底部。
由于关于当地文化、自然和环境的问题如此之少，而关于政府机构和经济的问题如此之多，一种相当循环的推理变得明显起来，即不断加强对语言使用的制度和政治方面的重视，而越来越忽视独特的传统、文化和环境方面的重要性。这可能会进一步巩固“语言分层模式，使那些不掌握标准的群体处于从属地位”，并强调那些使用主要的、政治和经济上有价值的、“标准”语言的人的利益。这种等级意识自动地将许多少数民族语言，如村庄手语，放在等级的最底层。
# 7.结论
这篇论文考虑了一系列关于手语濒危和生命力的工作背后的意识形态，这些工作导致了联合国教科文组织第一次将手语和口语并列纳入调查。在这种研究中，对意识形态的考察在评估工作成效时是有用的。突显了在濒危意识形态中构建手语的问题，并表明一些学者对“语言”的看法与社区成员的看法截然不同，他们试图在研究中代表社区成员的声音。一些受访者参加了改编后的手语调查，他们既是学者，也是手语社区的成员，我们希望这一领域的聋人学者的数量将继续增长。定量研究这一复杂现象为更有力地评估个别语言提供了一个起点。为了反映语言社区的独特情况，评分过程应该更多地考虑社区自身的观点，并辅之以混合方法的评估，可能会有加权因素。
