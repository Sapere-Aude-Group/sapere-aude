# 1.研究背景和目的
在耳语中，发声时的周期性语音源被噪声源取代，因此，话语中没有基频。这就给主要线索声调和语调带来了问题。因此，语音信号中必须存在非F0提示，以弥补耳语中基频的缺失。然而，尚未清楚证明的是，非F0线索是只为耳语产生的，还是它们已经存在于发声话语中，但在没有F0的情况下变得更有用。第一种机制可以被称为生产增强解释，而第二种机制可以被称为知觉补偿解释。说话者根据发音增强理论，制定了特殊的发音策略，以弥补F0的缺失，从而帮助听话者感知耳语中的语调和语调。如许多研究所暗示的那样，说话者或暗示或明确地修改某些非旋律属性以补偿旋律线索的丢失。相比之下，知觉补偿解释假设非音高的声调提示已经存在于发声语音中，但它们只有在没有F0的情况下才变得更有用，在耳语的情况下才会变得更有用在知觉补偿解释中，非F0线索的相关性增加只被视为一种知觉现象。
因此，这两种说法的关键不同之处在于，耳语中的非F0提示是否来自说话者为弥补基频缺失而开发的特殊发音动作。这些动作要么在发声话语中没有出现，要么在耳语中被夸大了。此外，这些动作不是为了产生耳语本身，而是为了在没有F0的情况下直接促进音调感知。本研究是对汉语耳语的进一步探索，旨在确定说话者是否形成了特殊的声学线索来帮助感知语言中的声调和语调。为此，我们进行了一次生产实验和三次知觉实验。
# 2.材料和方法
## 2.1发音实验
### 2.1.1被试
12名以普通话为母语的学生(平均年龄20.3岁，女性6名)，他们被分成6对，每对一男一女。
### 2.1.2刺激
表1中的所有字符都被用作发音刺激物。在有声和低语两种情况下，每个字符都被说成是一个陈述和一个疑问语气。总体而言，每个说话者总共有80个刺激(5个音节*4个声调*2个语调*2种发音类型)
![表1](../Supporting_Information/2021-05-23-LC1-Table-1.png)
### 2.1.3步骤
对说话者并排坐在实验室的录音室里，目标字符和对应的拼音出现在电脑屏幕上。被要求表演一段对话，其中一个人说这个字是一个疑问句，另一个人说的是同一个字作为回答。然后，在下一个角色中，两人互换了角色，第二个说话者成为了提出疑问的人。表2给出了说明。对于发声和耳语的情况，输入音量被设置为相同的。实验者在电话亭外监视和控制录音的进度。

## 2.2知觉实验
### 2.2.1被试
22名母语为汉语的学生(平均年龄：20.2岁，12名女性)。
### 2.2.2刺激
表1中的三个元音列(/ɤ/、/i/和/y/)中的发音和耳语字符被用作前两个实验的刺激(从原始语音中辨别声调和语调)。对于第三个实验(调幅噪声中的音调识别)，这些自然标记被用作产生调幅噪声的基础。编写了Praat(Boersma，2001)脚本，以首先提取基础刺激的幅度层，同时还保留了基础刺激的持续时间。然后将振幅分布施加到相同持续时间的粉色噪声上(在脚本中从50Hz开始，过滤-6dB/八度去加重滤除白噪声来生成)。以这种方式生成的声音保留了原始的持续时间和振幅剖面，但没有频谱信息。对于发声音节和耳语音节都产生了调幅噪声。在这个过程中，最大绝对振幅被调整到0.9，这抵消了有声和耳语在整体强度上的差异。总之，每个听者有96次试验(3个音节*4个声调*2个语调*2种发音类型*2次重复)。
### 2.2.3步骤
被试在安静的房间里参加了知觉实验。这些测试是使用PRAAT通过实验MFC脚本运行的。受试者戴着Sennheiser PC 166耳机，坐在电脑(OptiPlex390)前。在每个实验中，受试者听到一个声音(或噪音)，并在屏幕上看到相应音节的四个汉字和四个声调。然后，他们选择与他们所听到的最接近的字符。每个刺激只播放一次。每个受试者大约需要30-40分钟才能完成。所有受试者都先用自然语音刺激(声调和语调识别)进行测试，然后用调幅噪声(声调识别)进行测试。对于自然语言任务，大约一半(10)的人先听到了耳语的语块，而另一半(10)则先听到了发声的语块。在每个区块内，刺激物都是随机的。而且每个参与者的随机化都是不同的

表4列出了四向重复测量方差分析的重要结果。与有声发声相比，耳语具有更长的持续时间(508.743比384.664毫秒)，更弱的强度(41.056比66.005分贝)，更高的重心(1228.753比547.657赫兹)，更小的0-2千赫和2-5千赫之间的能量差(汉马堡指数：11.573比21.24dB)，低于500HZ(0.349比0.679dB)和低于1000HZ(0.543比0.858 dB)的能量更少，更高的F1(779.443比511.038 Hz)、F2(1821.279 比1524.67 Hz)和F3(3105.766比2759.81 Hz)。

### 3.1.2声调效应及其与发声的相互作用

图3显示了发声和耳语中四种声调强度的平均时间分布。可以看出，对于每个音调。这里的图3进一步显示，在耳语中没有产生音调的强度和持续时间的增强提示。

### 3.1.3语调的效果及其与语音的相互作用
图7显示，在耳语中，两种语调的COG都高于发声话语，陈述和疑问句之间的差异也大于发声话语。这可能是一种潜在的增强，可以增加耳语中的疑问和陈述之间的对比。
![图7](../Supporting_Information/2021-05-23-LC1-Fig-7.png)
图8显示，耳语中的语调具有较小的Hammarberg指数(图8a)和1000Hz以下的能量(图8b)，这两个值都表明比语音中更平坦的频谱倾斜。此外，疑问的频谱斜率比耳语中陈述的频谱斜率平坦得多。因此，这些都是潜在增强的进一步迹象。
![图8](../Supporting_Information/2021-05-23-LC1-Fig-8.png)
综上所述，就语调效果而言，耳语中的疑问句有更大的COG，但在1000Hz以下的Hammarberg指数和能量较低，都表明语谱斜率比陈述句中的平坦。这些是语调的潜在声学提示。但它们的有效性需要理性地评估。

# 4. 讨论和结论
生产实验检查了持续时间、强度、频谱倾斜和共振峰的9个测量值。对于声调，所有这些测量结果都没有显示出增强的证据，只有F2可能是例外。然而，经过更仔细的检查，即使是F2在耳语中的特殊行为也被确定为可能是由于共振峰跟踪错误而不是真正的增强。语调方面，耳语中疑问句的COG较大，而Hammarberg指数和1000 Hz以下的能量较低，均表明语音斜率较陈述句平缓。
因此，目前的数据几乎没有提供任何证据表明，耳语普通话的声调或语调都有有效的提高。
